# Sheikh Mohammad Ahmed — Portfolio Website

Personal portfolio website built for the **Nexsoft Solutions** internship program. A premium, responsive, single-page portfolio with dark/light theme toggle.

## Features

- ✨ Modern gradient design with glassmorphism effects
- 🌗 Dark/Light theme toggle (persisted via localStorage)
- 📱 Fully responsive (mobile, tablet, desktop)
- 🎯 Project filtering (Hackathons / Companies / Clients)
- 🔥 Scroll-triggered animations
- 🖱️ Custom cursor on desktop
- 📄 Downloadable resume

## Tech Stack

Vanilla HTML5, CSS3, JavaScript (ES6+) — no frameworks, no build tools.
